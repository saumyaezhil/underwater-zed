Underwater ZED 2i Capture

Features
- RGB + Depth + Point Cloud capture
- Underwater enhancement
- Toggle save with 's'
- Quit with 'q'

Saved format:
rgb_*.png
depth_*.npy
pc_*.npy

Command to run: 
```
python3 underwater_zed.py
```

Command to run if cloning on to monitor: 
```
-git clone https://github.com/YOUR_USERNAME/underwater-zed.git
-cd underwater-zed
```
```
pip3 install -r requirements.txt
```
```
python3 underwater_zed.py
```
